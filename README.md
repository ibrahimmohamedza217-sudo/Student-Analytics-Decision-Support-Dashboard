# 🎓 Student Analytics & Decision Support Dashboard

An interactive bilingual (Arabic & English) web-based analytics dashboard designed to monitor student performance, engagement, and retention. The dashboard transforms raw educational data into actionable insights through dynamic KPIs, interactive visualizations, and comprehensive reports to support data-driven decision making.

## 🚀 Features

- 📈 Student Performance Analysis
- ⚠️ At-Risk Student Detection
- 📁 Excel & CSV Data Import
- 🔍 Advanced Filtering & Search
- 📋 Dynamic Reports
- 🌍 Arabic & English Support
- 📱 Responsive Design
- 📊 Interactive Charts & Visualizations

## 🛠️ Technologies Used

- HTML5

- JavaScript
- Chart.js

## 📌 Project Objective

This project was developed to help educational organizations monitor student performance, identify at-risk students, analyze engagement trends, and support strategic decision-making through an intuitive analytics dashboard.

---

## 🎯 Key Capabilities

- Monitor student performance
- Analyze academic trends
- Track engagement and retention
- Generate visual reports
- Import and analyze Excel datasets
- Support educational decision-making
